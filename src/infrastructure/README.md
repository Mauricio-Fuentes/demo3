# Capa Infrastructure

Aquí van las **implementaciones técnicas** que dependen de frameworks o servicios externos:

- Implementaciones de repositorios (TypeORM, Prisma, Mongo, etc.)
- Adaptadores a servicios externos (APIs, colas, email, etc.)
- Configuración de módulos de infraestructura (ORM, cache, etc.)

Debe implementar las interfaces definidas en `domain`.
