# Capa Interfaces (o API / Presentation)

Responsable de exponer la aplicación al exterior:

- Controladores HTTP (REST, GraphQL, WebSockets, etc.)
- DTOs de entrada/salida expuestos a los clientes
- Mapeo entre DTOs y los modelos de `application` / `domain`

Aquí es donde se declaran los controladores propios de NestJS.
